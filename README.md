# Akademiya Cizîrî

Online Kurdish academy — language, history, geography, technology, and character education for Kurdish learners worldwide.

**Live site:** https://akademiyaciziri.com

## About

Akademiya Cizîrî is a project of the **Kurdish American Solidarity Association (KASA)** — https://kasaus.org. The academy offers free instruction in:

- **Languages:** Kurmancî, Soranî (coming), Zazakî (coming), English
- **Kurdish History & Geography**
- **World History from a Kurdish perspective**
- **Computer Science, IT, and Coding**
- **Character Education** (Trust, Respect, Responsibility, Fairness, Caring, Citizenship)

For all Kurdish people worldwide — *ji zarokan heta mezinan* (from children to adults).

## Tech stack

- **CMS:** WordPress
- **Theme:** Astra child theme (custom — see `theme/`)
- **LMS:** Tutor LMS
- **Donations:** GiveWP
- **Builder:** Elementor (templates in `wordpress/elementor-templates/`)
- **Automation:** Python scripts in `scripts/` (WP REST API)
- **Hosting:** Hostinger

## Repository structure

```
theme/         Custom Astra child theme + page templates (page-*.php)
wordpress/     Brand assets, humans.txt, Elementor templates, playground blueprint
scripts/       Python WordPress automation (content upload, fixes, replacements)
images/        Site imagery (hero, mascots, age groups, sections)
qerekter-girine/  Character education content
raporlar/      Internal meeting notes
```
